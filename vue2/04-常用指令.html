vue常用指令 ​

vue的指令是dom的行内属性,vue给这类属性赋予了一定的意义, 实现特殊的功能,所有的指令以v-开头

当表达式的值改变时，DOM也会响应式的产生变化。

1.v-text指令 ​

1.1 指令基本用法 ​

1.v-text指令不仅能将data中的数据渲染到页面上,还能对数据进行运算

2.v-text会将标签中的内容替换

js
<div id="root">
      <!-- 将data中的数据渲染到页面上 -->
      <div v-text="msg"></div>
      <!-- 对数据进行运算并渲染 -->
      <div v-text="msg + 'world'"></div>
    </div>
    <script>
      const vm = new Vue({
        el: "#root",
        data() {
          return {
            msg: "hello",
          }
        },
      })
    </script>

注意:

v-text指令的属性值虽然被引号包裹,但并不是字符串,而是表达式.

属性值也可以写成字符串,但要注意与外面包裹引号的冲突

js
<div v-text="msg + 'world'"></div> //这是数据与字符串的拼接

1.2指令会覆盖标签中的内容 ​

v-text指令的标签内最好不要写内容,因为会被覆盖

js
<div id="root">
      <!-- 你好不会显示,被msg数据覆盖 -->
      <div v-text="msg">你好</div>
    </div>
    <script>
      const vm = new Vue({
        el: "#root",
        data() {
          return {
            msg: "hello",
          }
        },
      })
    </script>

2.v-cloak指令 ​

插值符号在页面刷新的时候,会出现在页面上, 插值表达式会闪烁, v-cloak 解决闪烁问题

​ 1.给元素添加v-cloak

js
<div v-cloak></div>

3.v-html指令 ​

双大括号会将所有数据识别为文本，而不是html结构,为了输出真正的 HTML结构，需要使用 v-html 指令.

3.1 v-html指令的基本使用

把html字符串当成html结构渲染

js
<div id="app">
    <!-- v-text无法解析数据中h2标签 -->
    <div v-text="msg"></div>

    <!-- v-html可以解析数据中h2标签 -->
    <div v-html="msg"></div>

</div>

<script>
    var vm = new Vue({
        el: '#app',
        data: {
            msg:"<h2>hello wrold</h2>"
        }
    })
</script>

使用v-text指令的div并没有解析出数据中的h2标签,使用v-html指令的div,将数据中的h2解析成了标签

4.v-once指令 ​

我们使用Mustache语法,包括v-text,v-html指令渲染的页面都是具有响应式的, 当数据发生变化, 视图也会跟着改变

v-once属性不用设置值,使用插值符号插值,v-once只会初始化一次;当数据发生变化时,页面视图不会发生改变

js
<div id="app">
    <!-- 使用v-once指令 -->
    <div v-once>{{msg}}</div>

    <!-- 未使用v-once指令 -->
    <div>{{msg}}</div>
</div>

<script>
    const vm = new Vue({
        el: "#app",
        data: {
            msg: 'hello wrold'
        }
    })

    // 3秒以后数据发生变化,页面视图没有改变,
    setTimeout(() => {
        vm.msg = "哈哈哈 "
    }, 3000)
</script>

5.v-bind指令 ​

v-bind指令会将普通属性的值变成表达式

5.1基本使用 ​

js
<div id="root">
   	  //使用v-bind指令绑定class属性
      <h1 v-bind:class="msg"></h1>
	  //未使用v-bind指令
      <h2 class-="box"></h2>
    </div>
    <script>
      const vm = new Vue({
        el: "#root",
        data() {
          return {
            msg: "className",
          }
        },
      })
    </script>

5.2简写形式 ​

js
//简写形式
      <h1 :class="msg"></h1>
	  //完整格式
      <h2 v-bind:class-="msg"></h2>

6.v-on事件监听 ​

6.1基本使用 ​

button绑定点击事件,被点击时,就会触发methods属性中的的方法, 然后改变数据,数据的改变同时会触发视图的重新渲染,

js
<div id="root">
      <h2>{{msg}}</h2>
      <button v-on:click="change">点我改变</button>
    </div>
    <script>
      const vm = new Vue({
        el: "#root",
        data() {
          return {
            msg: "hello",
          }
        },
        methods: {
          change() {
            this.msg = "你好"
          },
        },
      })
    </script>

6.2事件绑定的简写形式 ​

js
<!-- 完整语法 -->
<button v-on:click="change">点我改变</button>

<!-- 缩写 -->
<button @click="change">点我改变</button>

6.3注意函数内this指向的问题 ​

普通方法中this指向Vue 实例对象

js
<div id="app">
    <!-- 绑定点击事件触发Vue中的reversed方法 -->
    <button @click="reversed">按钮</button>
</div>

<script>
    let vm = new Vue({
        el: "#app",
        data: {
            msg: "Hello"
        },
        methods: {
            reversed() {
                console.log(this) // 实例对象
            }
        }
    })
</script>

将实例中的方法换成箭头函数

js
<div id="app">
    <button @click="reversed">按钮</button>
</div>

<script>
    let vm = new Vue({
        el: "#app",
        data: {
            msg: "Hello"
        },
        methods: {
            reversed:()=> {
                console.log(this) // 实例对象
            }
        }
    })

</script>

如果换成箭头函数,this 将不再是实例对象,而是window.

ethods属性中的方法不推荐使用箭头函数,但也有特殊情况下需要使用箭头函数,例如下面的示例:

js
new Vue({
    el:"#app",
    data:{
        msg:"Hello Vue"
    },
    methods: {
        reversed(){
            console.log(this)

            let fn = () => {
                console.log(this)// Vue实例对象
            }

            fn()
        }
    }
})

如果要在reversed 里面的函数 fn 里,使用实例对象就需要使用箭头函数了.如果使用普通函数, 使用添加括号的普通调用方式,那么fn函数中的this将不再是Vue实例对象

6.4关于方法的事件对象 ​

调用方法的时候有两种情况,一种是不加括号,一种加括号

加括号的方式是为了传递参数给事件函数

js
<!--不加括号 -->
<button @click="reversed">点击</button>

<!--加括号 -->
<button @click="reversed()">点击</button>
<button @click="reversed(12345)">点击</button>

6.4.1 不加括号 ​

不加括号的情况下,默认第一个形参就是事件对象

js
<div id="app">
    <!-- 不加括号绑定事件 -->
    <button @click="reversed">点击</button>
</div>

<script>
    let vm = new Vue({
        el: "#app",
        data: {
            msg: "Hello"
        },
        methods: {
            reversed:(ev)=> {
                console.log(ev) // 事件对象
            }
        }
    })

</script>

6.4.2 加括号 ​

加括号无论传不传实参,形参默认接受传递的形参

js
<div id="app">
    <!-- 加括号未传参 -->
    <button @click="reversed()">点击</button>

    <!-- 加括号传参 -->
    <button @click="reversed(123)">点击2</button>
</div>

<script>
    let vm = new Vue({
        el: "#app",
        data: {
            msg: "Hello"
        },
        methods: {
            reversed:(ev)=> {
                console.log(ev) // 事件对象
            }
        }
    })

</script>

加括号没有传参是,事件函数的第一个形参是undefined因为没有接受到实参

加括号传递了实参,那么事件函数的第一个形参就是传递过来的实参

6.4.3 绑定事件手动传递事件对象 ​

当传参时如何获取事件对象呢 如果加括号就需要手动的传递事件对象

js
<div id="app">
    <!-- 使用$event占位 -->
    <button @click="reversed($event,123)">点击</button>
</div>

<script>
    let vm = new Vue({
        el: "#app",
        data: {
            msg: "Hello"
        },
        methods: {
            reversed:(ev,num)=> {
                console.log(ev) // 事件对象
              console.log(num) // 传递的参数
            }
        }
    })

</script>

注意: 无论加不加括号,传不传参数,在函数里变量event默认是事件对象, 所以不要定义同名变量将其覆盖就可以了

js
<div id="app">
    <!-- 使用$event占位 -->
    <button @click="reversed($event,123)">点击</button>
</div>

<script>
    let vm = new Vue({
        el: "#app",
        data: {
            msg: "Hello"
        },
        methods: {
            reversed:(ev,num)=> {
                console.log(event) // 事件对象
            }
        }
    })
</script>

7. v-mode双向数据绑定 ​

v-model 指令: 它能轻松实现表单输入和存储数据之间的双向绑定

7.1基本使用 ​

js
<div id="app">
    <input type="text" v-model="msg">
    {{msg}}
</div>

<script>
    var vm = new Vue({
        el: '#app',
        data: {
            msg: 'hello'
        }
    })
</script>

用已学的知识实现v-model的功能

js
<div id="app">
  	  //使用v-model
      <input type="text" v-model="name" />{{name}}
      //模拟实现v-model功能
      <input type="text" ref="ipt" :value="msg" v-on:keyup="aa" />{{msg}}
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          name: "aa",
          msg: "hello",
        },
        methods: {
          aa(el) {
            console.log(el)
            this.msg = this.$refs.ipt.value
          },
        },
      })
    </script>

7.2 checkbox ​

checkbox是表单的多选按钮,如果多选按钮只有一个时

7.2.1在checkbox上使用v-model (多选按钮只有一个) ​

  1. v-model绑定的是字符串

    会将字符串转成布尔值,true为选中,false为未选中.

js
<div id="app">
      <input type="checkbox" v-model="name"  />{{name}}
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          name: "",
        },
      })
    </script>

  1. v-model绑定的是数组( 注意:此时没有value值!!!!!!! )

    选中时数组收集到的是null,未选中收集到的是空

js
<div id="app">
      <input type="checkbox" v-model="name"  />{{name}}
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          name: [],
        },
      })
    </script>

  1. v-model绑定的是数组,此时有value值

    选中时收集到的是value的值为[haha],未选中收集到的是空数组

    js
    <div id="app">
      <input type="checkbox" v-model="name" value="haha" />{{name}}
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          name: [],
        },
      })
    </script>
7.2.2在checkbox 使用v-model (多选按钮有多个) ​
js
<div id="app">
      <input type="checkbox" v-model="name" value="haha" />{{name}}
      <input type="checkbox" v-model="name" value="nihao" />
      <input type="checkbox" v-model="name" value="women" />
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          name: [],
        },
      })
    </script>

此时初始都是未选中,当选中按钮时,数组中就会收集对应value的值

7.4 select ​

v-model结合select使用,select也分为单选和多选

7.4.1 select单选 ​
js
<div id="app">
      <select name="fruit" v-model="fruit">
        <option value="苹果">苹果</option>
        <option value="梨子">梨子</option>
        <option value="香蕉">香蕉</option>
      </select>
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          fruit: "",
        },
      })
    </script>

​ 1.select双向绑定的数据是一个字符串,每次只收集一个数据,

​ 如果绑定的数据不是字符串,在选中内容后,数据也会被替换为字符串类型的数据

​ 2.就算你把数据中的fruit的值写成数组,当选中水果后,fruit值会被替换为字符串数组

​ 3.默认没有一个水果被选中,如果想要默认选中内容, 可以给fruit数据添加默认值

​ 1.添加了multiple属性,让下拉选择框变成多选

​ 2.双向数据绑定的数据要收集多个数据,此时绑定的数据属性值应该为数组

js
<div id="app">
      <select name="fruit" v-model="fruit" multiple>

        <option value="梨子">梨子</option>
        <option value="香蕉">香蕉</option>
      </select>
      <h2>你选择的水果是:{{fruit}}</h2>
    </div>
    <script>
      var vm = new Vue({
        el: "#app",
        data: {
          fruit: [],
        },
      })
    </script>

8 判断指令 ​

8.1v-if ​

v-if指令是用来控制元素的切换显示

js
<div id="app">
      <h2 v-if="seen">你好哈哈哈</h2>
    </div>
    <script>
      const vm = new Vue({

8.2v-else ​

v-else 指令来表示 v-if 的“else 块”

js
<div v-if="Math.random() > 0.5">
  你能看到我
</div>
<div v-else>
   你看不到我
</div>

v-else 元素必须紧跟在带 v-if 或者 v-else-if 的元素的后面，否则将不会被识别。

8.3v-else-if ​

js
<div id="app">
      <h2 v-if="aa">你好</h2>
      <h2 v-else-if="bb">吃饭了吗</h2>

      <button @click="changeAa">更改a</button>
      <button @click="changeBb">更改b</button>
    </div>
    <script>
      //实例化Vue对象
      const vm = new Vue({
        el: "#app",
        data: {
          aa: false,
          bb: false,
        },
        methods: {
          changeAa() {
            this.aa = !this.aa
          },
          changeBb() {
            this.bb = !this.bb
          },
        },
      })
    </script>

8.4使用template标签 ​

template标签, 是vue提供给我们的没有实际意义的,用来包裹元素的,

如果我需要多个标签同时显示或隐藏,我们就可以把这些标签包裹在同一标签里

js
<div id="app">
      <template v-if="cut">
        <label>注册</label>
        <input type="text" />
      </template>
      <template v-else>
        <label>登录</label>
        <input type="text" />
      </template>

    </div>
    <script>
      const vm = new Vue({
        el: "#app",
        data: {
          cut: false,
        },
      })
    </script>

9 v-show ​

9.1基本使用 ​

  1. v-show 不支持 <template> 元素，

  2. v-show 指令不支持 v-else。使用会报错

    js
    <template v-show="false">
    <div>你好</div>
    <div>你好</div>
    <div>你好</div>
    <div>你好</div>
</template>

    虽然v-show是false,但是template里面的div依然会显示在网页上,同时控制台并没有报错.

9.2v-if VS v-show ​

v-if判断为假时，整个元素都会从DOM节点上删除

v-show 的元素始终会被渲染并保留在 DOM 中。v-show 只是简单地切换元素的 CSS 属性 display,元素即使不显示也依然在DOM节点上占位。

10 v-for ​

10.1数组的循环 ​
js
<div id="app">

数组项为对象的循环

js
<div id="app">
      <ul>
        <li v-for="item in fruites">
          <span>{{item.name}}:</span>
          <span>{{item.price}}</span>
        </li>
      </ul>
    </div>

    <script>
      const vm = new Vue({
        el: "#app",
        data: {
          fruites: [
            {
              name: "苹果",
              price: "8元/斤",
            },
            {
              name: "香蕉",
              price: "4元/斤",
            },
            {
              name: "梨子",
              price: "7元/斤",
            },
            {
              name: "荔枝",
              price: "15元/斤",
            },
          ],
        },
      })
    </script>

注意: 我们也可以用of替代in作为分割符

10.2对象的循环 ​

用 v-for 指令来循环对象

第一个参数是循环对象的属性值:(value)

第二个参数是循环对象的属性名:(key)

第三个参数是循环对象的索引值:(index)

js
<div id="app">
      <ul>
        <li v-for="(value,key,index) in users">
          {{index}}. {{key}} : {{ value }}
        </li>
      </ul>
    </div>

    <script>
      const vm = new Vue({
        el: "#app",
        data: {
          users: {
            name: "Henry",
            age: 22,
            work: "前端工程师",
            like: "看书",
          },
        },
      })
    </script>
10.3v-for循环中的key属性 ​

使用v-for更新已渲染的元素列表, 列表数据修改的时候, 会根据key值去判断某个值是否修改,如果修改,则重新渲染这一项,否则复用之前的元素; 我们经常会使用index(即数组的下标)来作为key,但这是不推荐的一种使用方法 ,key值的使用其实是和vue响应式的虚拟DOM有关.

  1. 初始数组循环显示
js
<div id="app">
      <div v-for="(item, index) in fruites" :key="index">{{item.name}}</div>
    </div>
    <script>
      new Vue({
        el: "#app",
        data: {
          fruites: [
            {
              id: 1,
              name: "苹果",
            },
            {
              id: 2,
              name: "梨子",
            },
            {
              id: 3,
              name: "西瓜",
            },
          ],
        },
      })
    </script>

2.此时数据变化

js
data: {
          fruites: [
            {
              id: 1,
              name: "苹果",
            },
            {

如果数据是这种变化, 那么index没有问题

3.我们是在数组中间插入数据就会有问题了

js
<div id="app">
      <div v-for="(item,key, index) in fruites" :key="index">
        <span>{{item.name}}</span>
        <span>{{key}}</span>
        <span>{{index}}</span>
      </div>
    </div>
    <script>
      new Vue({
        el: "#app",
        data: {
          fruites: [
            {
              id: 1,
              name: "苹果",
            },
            {
              id: 4,
              name: "我是新增的水果",
            },
            {
              id: 2,
              name: "梨子",
            },
            {
              id: 3,
              name: "西瓜",
            },
          ],
        },
      })
    </script>

除了第一个数据可以复用之前的之外,另外三条数据都需要重新渲染

4.我们可以使用数组中不会变化的那一项作为key值

js
<div id="app">
      <div v-for="(item,key, index) in fruites" :key="id">
        <span>{{item.name}}</span>
        <span>{{key}}</span>
        <span>{{index}}</span>
      </div>
    </div>

建议尽可能在使用 v-for 时提供 key
